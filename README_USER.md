# Place-DB Pro 사용자 가이드

Place-DB Pro는 네이버 플레이스 데이터를 수집하여 로컬 DB에 저장하고 엑셀로 추출할 수 있는 독립 실행형 마케팅 툴입니다.

## 1. 프로그램 실행 방법 (매우 간단!)
1. 전달받은 폴더의 압축을 풉니다.
2. **Place-DB-Pro-실행.bat** 파일을 더블 클릭하여 실행합니다.
   - *참고: 이 파일은 실행 환경을 점검하고 프로그램을 안전하게 시작합니다.*
3. 잠시 기다리면 프로그램 메인 화면이 나타납니다.

## 2. 보안 인증 (최초 1회)
1. 프로그램을 시작하면 '인증 창'이 나타납니다.
2. **내 PC ID** 항목에 표시된 코드를 복사하여 관리자에게 전달하고 **시리얼 번호**를 발급받으세요.
3. 발급받은 시리얼 번호를 입력하고 '인증하기'를 누르면 모든 기능을 사용할 수 있습니다.

## 3. 데이터 수집 및 추출
1. **검색 키워드**와 **지역**을 입력합니다.
2. **크롤링 시작** 버튼을 누르면 데이터 수집이 시작됩니다.
3. 수집이 끝나면 **엑셀 내보내기** 버튼을 눌러 `exports` 폴더에 저장된 파일을 확인하세요.

## 4. 문제 해결 (에러 발생 시)
- **실행이 안 될 경우**: 
  - `dependencies` 폴더 안의 `vc_redist.x64.exe` 파일을 설치한 후 다시 실행해 보세요.
  - 프로그램 폴더를 `C:\Place-DB`와 같이 영문 중심의 간단한 경로로 옮겨서 실행해 보세요.
- **백신 차단**: 일부 백신이 오진하는 경우 실행 파일을 '예외 목록'에 추가해 주세요.

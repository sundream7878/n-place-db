# Place-DB Pro & Basic 프로젝트 통합 계획 및 현황

이 문서는 Place-DB 프로젝트의 현재 기능 명세와 지금까지 진행된 주요 업데이트 및 향후 계획을 정리한 문서입니다.

## 1. 프로젝트 개요
네이버 플레이스 데이터를 수집하여 마케팅 자료로 활용할 수 있도록 돕는 독립 실행형 자동화 툴입니다. 사용자 편의성을 극대화한 '무설치' 배포 방식을 지향합니다.

## 2. 주요 기능 (Key Features)

### 🚀 배포 최적화 (Distribution)
- **무설치 실행 (DLL Bundling)**: PyInstaller의 바이너리 수집 기능을 강화하여 `vcruntime140.dll` 등 필수 런타임을 EXE에 내장. 구매자가 별도로 Visual C++를 설치하지 않아도 실행 가능하도록 구현.
- **자동화된 배포 준비**: `prepare_dist.py`를 통해 로그, 개인 데이터베이스, 임시 파일을 일괄 삭제하고 최종 배포 파일을 구성하는 프로세스 확립.
- **스마트 런처**: 시스템 환경을 자동으로 점검하고, 에러 상황 시 사용자 가이드를 제공하는 `.bat` 기반의 원클릭 실행기 제공.

### 🛠 버전별 차별화 (Version Control)
- **Place-DB Pro**: 전체 기능(데이터 수집 + 메신저 연동 + 정기 관리 등) 포함 버전.
- **Place-DB Basic**: 핵심 데이터 수집 및 엑셀 내보내기 전용 라이트 버전.

### 🎨 디자인 및 UI/UX
- **Premium Dark Mode**: 다크 테마 기반의 세련된 UI 적용.
- **브랜딩 최적화**: 실행 파일 및 런처 이름을 직관적으로 변경 (`Place-DB-Pro.exe`, `Place-DB-Basic.exe`).

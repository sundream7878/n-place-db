# N-Place-DB 문제 해결 가이드 (DLL 에러)

프로그램 실행 시 **"Failed to load Python DLL"** 또는 **"지정된 모듈을 찾을 수 없습니다"** 에러가 발생하는 경우 아래 단계를 순서대로 진행해 주세요.

### 1단계: 필수 런타임 설치 (가장 중요)
Python 프로그램을 실행하기 위해 필요한 Microsoft 공식 라이브러리가 PC에 없을 때 발생합니다.
- **다운로드 링크**: [Microsoft Visual C++ Redistributable (x64)](https://aka.ms/vs/17/release/vc_redist.x64.exe)
- 위 링크를 클릭하여 파일을 다운로드한 후 설치를 완료하고 PC를 다시 시작해 보세요.

### 2단계: 프로그램 설치 경로 변경
프로그램이 들어있는 폴더의 경로에 한글이나 특수문자가 포함되어 있으면 DLL을 불러오지 못하는 경우가 있습니다.
- **현재 경로 예시 (에러 발생 가능)**: `C:\Users\사용자\Desktop\수집기`
- **권장 경로 (정상 작동)**: `C:\N-Place-DB`
- 폴더 전체를 `C:\` 드라이브 바로 아래로 옮겨서 실행해 보세요.

### 3단계: 백신 프로그램 예외 처리
일부 백신 프로그램(V3, 알약 등)이 실행 파일을 차단하는 경우가 있습니다.
- 실행 파일(`N-Place-DB.exe`)을 백신 예외 목록에 추가하거나, 잠시 실시간 감시를 끄고 실행해 보세요.

---
**주의**: 본 프로그램은 Windows 10/11 64비트 환경에서만 정상 작동합니다.

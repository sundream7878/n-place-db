# 제목: N-Place-DB Pro 기능명세서 (Feature Specification)
- **버전**: v1.1.46
- **일시**: 2026-05-18 14:42

---

## 1. 개요 (Introduction)
N-Place-DB Pro는 로컬 네이버 플레이스 데이터를 초고속으로 크롤링하고 수집된 정보를 바탕으로 이메일 마케팅 및 인스타 DM 마케팅을 통합 수행할 수 있는 올인원(All-in-One) 로컬 마케팅 솔루션입니다. 

---

## 2. 모듈별 세부 기능 명세

### A. 플레이스 데이터 수집 엔진 (Naver Place Crawler)
*   **멀티셀렉트 정밀 지역 설정**: 
    - 대한민국 모든 시/도 및 시/군/구 상세 행정구역에 대한 다중 선택(Multi-select) 수집 방식을 지원합니다.
*   **실시간 중복 스킵 인덱싱 (Smart Skip)**: 
    - SQLite 로컬 DB 구조를 활용하여 이미 수집된 업체(동일 `detail_url` 기준)는 네트워크 요청 전에 실시간으로 건너뛰어 속도 향상 및 중복 수집을 차단합니다.
*   **통합 수집 대시보드**:
    - **예상 대상 수**: 초기 탐색 후 예상 수집 대상 건수를 계산하여 표시합니다.
    - **실시간 소요 시간 및 평균 처리 속도(초/건)**를 동적으로 리포트합니다.
    - **실시간 수집 현황 테이블**: 수집된 최신 600개의 데이터셋이 표에 즉각 업데이트됩니다.
    - **실시간 모니터링 로그창**: 백그라운드 엔진의 전체 로그가 실시간 연동되어 스크롤 박스로 모니터링 가능합니다.
*   **엑셀(CSV) 내보내기**: 
    - `utf-8-sig` 인코딩을 적용해 한글 깨짐 현상 없이 언제든 원클릭으로 엑셀 호환 CSV 파일을 저장 및 다운로드할 수 있습니다.

### B. 이메일 마케팅 발송 엔진 (Email Sender)
*   **다양한 SMTP 규격 대응**: 
    - Naver, Gmail, Kakao 등 주요 포털 메일 서버의 **SSL(465) 및 STARTTLS(587)** 보안 인증 규격을 완벽 지원합니다.
*   **발신자 이름 히스토리 커스터마이징**:
    - 발신자 명의를 동적으로 입력 가능하며, 한 번 입력한 명칭은 자동으로 데이터베이스에 보존되어 다음 발송 시 드롭다운 선택만으로 손쉽게 지정할 수 있습니다. (새 명칭 등록 / 삭제 기능 포함)
*   **본문 동적 태그 치환**:
    - 본문에 `{상호명}` 예약 태그 입력 시, 수집된 업체의 실제 상호명으로 자동 치환 발송됩니다.
*   **테스트 발송 가상 로딩**:
    - 수집 결과 리스트 맨 위에 사용자가 지정한 테스트 수신인 계정(최대 3개)을 시각적으로 배치하여, 본 발송 실행 전 정상 작동 여부를 사전에 검증할 수 있습니다.

### C. 인스타 DM 마케팅 발송 엔진 (Instagram DM Automator)
*   **Playwright 웹 자동화 제어**:
    - 실제 크롬(Chrome) 브라우저 제어 기술(Playwright Stealth)을 사용해 로그인 및 발송 전 과정을 자동 수행합니다.
*   **다중 계정 및 쿠키 세션 영구 보존**:
    - 발송용 인스타 계정을 여러 개 추가 및 등록할 수 있으며, 한 번 성공한 로그인 쿠키 세션은 암호화된 JSON 파일로 영구 보존되어 다음 실행부터 비밀번호 재입력 없이 자동 진행됩니다.
*   **실시간 서브프로세스 로그 피드**:
    - 크로스오버 통신 딜레이를 제거하고 인스타 제어 백그라운드 프로세스의 모든 텍스트 로그(로그인, 타겟 매칭, 발송 완료)를 UI 모니터에 초 단위로 지연 없이 출력합니다.
*   **인스타 404/존재하지 않는 계정 자동 탐지**:
    - 폐쇄되거나 삭제된 인스타 계정 혹은 비공개 제한 계정에 대해 Playwright가 에러를 사전 감지하고, 브라우저가 정지하지 않고 `존재하지 않는 계정` 로그를 기록한 뒤 즉시 다음 타겟으로 유연하게 넘어갑니다.

### D. 라이센스 보안 및 정보 관리
*   **Supabase 기반 클라우드 인증**:
    - 사용자 하드웨어 고유 ID(HWID) 기반 시리얼 키 자동 바인딩으로 정교한 라이센스 기간 및 권한 통제가 실시간으로 작동합니다.
*   **바이트 단위 데이터 암호화 (XOR)**:
    - 로컬에 보존되는 민감 정보(이메일 비밀번호, 인스타 비밀번호)는 바이트 레벨의 XOR 알고리즘으로 즉각 암호화되어 외부에 투명하게 노출되지 않습니다.
